# Team Task Manager

## Description
A full-stack web application to manage projects and assign tasks with role-based access control.

## Features
- User Authentication (Login/Register)
- Role-based access (Admin / Member)
- Create and manage projects
- Assign tasks to users
- Track task status (Pending / Completed)
- Dashboard view

## Tech Stack
- Backend: Flask (Python)
- Database: SQLite
- Frontend: HTML, Bootstrap

## How to Run Locally
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

## Deployment
Deployed using Railway.

## Project Flow
User → Login → Dashboard → Create Project → Assign Task → Update Status
